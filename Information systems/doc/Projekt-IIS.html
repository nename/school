Use case

Implementace

Použité technologie

PHP 7.2, MySQL (SQL), framework Laravel, Composer, HTML, CSS

Autentizace

Vytvořená pomocí Laravelu. (php artisan make:auth)

Stárá se jak o registraci a přihlašování, tak i o znovuzískání hesla.

MatchController.php

Kontroler spravuje stránku zápasu.

Zobrazuje konkrétního stránku zápasu, vytváří nové zápasy, aktualizuje zápasy při změně (zadávání výsledků) a maže zápasy.

ProfileController.php

Kontroler spravuje uživatele.

Smazání uživatele (admin má práva), úprávu informací uživatele.

SearchController.php

Kontroler spravuje vyhledávací stránku.

Ukazuje všechny výsledky vyhledávání pro hráče, týmy a turnaje.

TeamController.php

Kontroler spravuje stránku týmů.

Zobrazuje konkrétní stránku týmu, vytváří týmy, maže týmy a edituje týmy (změna loga, povýšení hráče na vlastníka, opustit tým, přidat člena, vyhodit člena).

TournamentController.php

Kontroler spravuje stránku všech turnajů a stránku jednotlivých turnajů.

Na stránce všech turnajů je zobrazuje.

Vytváří a edituje turnaje a obstarává všechny akce pro uživatele jako jsou: vstoupit do turnaje, opustit turnaj, stát se rozhodčím, opustit turnaj jako rozhodčí, přidat a odebrat sponzora, vyhodit rozhočího, vyhodit tým a generuje zápasy spolu s pavoukem.

Databáze

ERD

Instalace

Stručně popište:
  • Jako host jsme zvolili digitalocean.com, což je cloudový hosting.
  • Server je Ubuntu LAMP 18.04.
  • Požadavky: PHP >= 7.2.0, BCMath, Ctype, JSON, Mbstring, OpenSSL, PDO, Tokenizer, XML (PHP extensions), Composer, MySQL, Apache (pro slibnou URL, na serveru běží Apache2)
  • Pro správné fungování databáze je nejlepší v MySql prvně vytvořit databázi samotnou. Následně vytvořit účet pro přístup aplikace a nastavit pravomoce. Dále je nutné zadat dané údaje do .env souboru, aby aplikace mohla přistupovat k databázi. Při úspěchu databázi vygenerujete příkazem "php artisan migrate" a naplníte "php artisan db:seed".

Známe problémy

V prohlížeči Mozila Firefox není podporovaný „datetime-local” typ inputu. Takže při výběru hodnoty se tu nenachází kalendář a tudíž formát vstupu není ošetřený.
